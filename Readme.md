## Description

Oh Hey! is a Dalamud plugin that has two main features currently:
- Notifying you when someone targets you
- Notifying you when someone emotes at you

The plugin notifies you with a text message (only visible to you) and can optionally play a notification sound.\
Additionally it tracks the most recent notification events with a simple interface.

## Installation
The plugin is available in the official repositories of [XIVLauncher](https://goatcorp.github.io/).
You can install it by opening the plugin installer in the game (`/xlplugins`) and searching for "Oh Hey!".
